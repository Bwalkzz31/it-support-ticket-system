# IT Support Ticket System

A web-based IT support ticketing system built with Flask, SQLite, and HTML/CSS.

---

## Features

- Submit support tickets
- Assign priority levels (Low, Medium, High)
- Mark tickets as resolved
- Dashboard showing:
  - Total tickets
  - Open tickets
  - Resolved tickets

---

## Technologies Used

- Python (Flask)
- SQLite
- HTML/CSS
